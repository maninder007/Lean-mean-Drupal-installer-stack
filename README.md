Recommended run flow


### 1. Log in via SSH key as your sudo user (e.g. feezix)
### 2. Create/edit ~/feesix-install.env (chmod 600)
### 3. Save script as feesix-installer.sh
### 4. chmod +x feesix-installer.sh
### 5. sudo ./feesix-installer.sh fresh
###    → first time: types password once
###    → after NOPASSWD applied: no more prompts ever
